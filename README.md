# Dublin Bikes Web App

Bike sharing web application with:

- station availability map
- weather data
- historical occupancy
- ML-based prediction

Developed for Software Engineering course using Scrum.

---

## Tech Stack

Backend:

- Python
- Flask

Frontend:

- HTML / JavaScript

Other:

- Web scraping
- MySQL
- Machine Learning

---

## Project Structure

```
dublin-bikes-app/
│
├── app/ # Flask backend (routes, templates, static)
├── scraper/ # Data collection scripts
├── models/ # ML models
├── docs/ # Personas, stories, mockups and so on
├── tests/
├── run.py # App entry point
├── environment.yml
├── README.md

```
